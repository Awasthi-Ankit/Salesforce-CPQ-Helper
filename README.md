# Salesforce CPQ Helper

## Overview
Salesforce CPQ Helper is a Chrome Extension built to assist Salesforce Developers and Admins with CPQ development, debugging, and migration â€” including moving from Salesforce CPQ to other tools like Salesforce RLM.  
It helps streamline tasks and save valuable development time.

## How to Use
- Install the extension.
- Open the Salesforce org tab where you are working.
- Use the extension within Salesforce tabs (it does not work on non-Salesforce pages).
